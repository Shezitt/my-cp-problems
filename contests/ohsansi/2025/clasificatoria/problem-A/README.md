# Problema A

Dado:
* $N$ $(1 \leq N \leq 10^{18})$

Imprimir:
* $N - N/2$

## Subtareas

### Subtarea 1 (20 puntos)
* N es par.

### Subtarea 2 (30 puntos)
* N es impar.

### Subtarea 3 (10 puntos)
* $1 \leq N \leq 10^5$

### Subtarea 4 (10 puntos)
* $1 \leq N \leq 10^9$

### Subtarea 5 (30 puntos)
* Sin restricciones adicionales.
